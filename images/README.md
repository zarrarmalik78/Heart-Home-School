# Image Assets Directory

This folder is organized to help you manage custom images for Heart Home School.

## How to use custom images:
1. Place your images in the appropriate subfolders.
2. Open `constants.tsx`.
3. Update the `IMAGES` object paths to point to your local files (e.g., `images/hero.jpg`).

## Folder Structure:
- `general/`: For background, hero, and section images.
- `programs/`: For images representing the different learning tiers.
- `teachers/`: For headshots of the educators.
- `testimonials/`: For parent photos.

## Recommended naming:
- `general/hero.jpg`
- `general/why-us.jpg`
- `programs/toddler.jpg`
- `teachers/fatima.jpg`
